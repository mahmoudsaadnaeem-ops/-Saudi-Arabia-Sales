# -Saudi-Arabia-Sales
A simple sales dashboard built to analyze sales performance. It shows total sales, average sales, monthly trends, product category performance, and sales distribution by city. The dashboard helps identify key insights and supports data-driven business decisions.  Tools: Power BI, Data Analysis, Data Visualizat
This project presents an interactive sales dashboard designed to analyze business performance and provide key insights into sales trends, product categories, and geographic distribution. The dashboard highlights total sales, average order value, and transaction counts while visualizing monthly sales trends, top-performing product categories, and sales distribution across cities.

The goal of this project is to demonstrate data analysis and data visualization skills by transforming raw sales data into meaningful insights that support business decision-making. The dashboard helps identify seasonal patterns, high-performing markets, and product demand trends.

Tools Used

Power BI

Data Visualization

Data Analysis

Business Intelligence

Key Insights

Identification of monthly sales trends and seasonality

Analysis of top product categories driving revenue

Geographic distribution of sales across major cities

Key performance indicators such as total sales, average sales, and number of transactions
